## 一、概念速查索引

### 1.1 基础概念

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **机器学习** | 让计算机从历史数据中自动学习经验、训练模型，从而对新数据做出预测或决策，无需显式编程。 | [[1.机器学习概述（AI增强版）]] |
| **样本** | 一行数据，也称一条记录，是数据的基本组成单位。 | [[1.机器学习概述（AI增强版）]] |
| **特征** | 一列数据，对预测有用的属性，构成特征向量。 | [[1.机器学习概述（AI增强版）]] |
| **标签/目标值** | 模型要预测的那一列，监督学习的核心依据。 | [[1.机器学习概述（AI增强版）]] |
| **训练集** | 用来训练模型的数据，让算法学习规律。 | [[1.机器学习概述（AI增强版）]] |
| **测试集** | 用来评估模型的数据，模拟真实场景中未见过的数据。 | [[1.机器学习概述（AI增强版）]] |
| **监督学习** | 输入数据由特征值+目标值组成，有标签的学习方式，分为分类和回归。 | [[1.机器学习概述（AI增强版）]] |
| **无监督学习** | 输入数据无标签，根据样本间相似性自动发现数据内在结构。 | [[1.机器学习概述（AI增强版）]] |
| **半监督学习** | 结合少量标注数据和大量未标注数据，通过伪标签迭代优化训练。 | [[1.机器学习概述（AI增强版）]] |
| **强化学习** | 通过智能体与环境的交互，根据奖励反馈学习最优策略。 | [[1.机器学习概述（AI增强版）]] |
| **特征工程** | 利用专业背景知识和技巧处理数据，提取、转换、选择特征，让算法效果最优。 | [[1.机器学习概述（AI增强版）]] |
| **特征提取** | 从原始数据中提取与任务相关的特征，构成特征向量。 | [[1.机器学习概述（AI增强版）]] |
| **特征预处理** | 将不同单位的特征转换到同一范围，消除量纲影响。 | [[1.机器学习概述（AI增强版）]] |
| **特征降维** | 降低数据维度，保留主要信息，如 PCA、特征选择。 | [[1.机器学习概述（AI增强版）]] |
| **特征选择** | 从众多特征中选出重要子集，简化模型、降低过拟合风险。 | [[1.机器学习概述（AI增强版）]] |
| **特征组合** | 把多个特征合并成一个新特征，如将身高和体重组合为 BMI。 | [[1.机器学习概述（AI增强版）]] |
| **建模六步法** | 获取数据 → 数据基本处理 → 特征工程 → 模型训练 → 模型评估 → 模型部署。 | [[1.机器学习概述（AI增强版）]] |
| **欠拟合** | 模型过于简单，训练集和测试集表现都差，没有学到规律。 | [[1.机器学习概述（AI增强版）]] |
| **过拟合** | 模型过于复杂，训练集表现好但测试集差，记住了训练集特有噪声。 | [[1.机器学习概述（AI增强版）]] |
| **泛化能力** | 模型在新数据上的预测能力，是衡量模型好坏的核心标准。 | [[1.机器学习概述（AI增强版）]] |
| **奥卡姆剃刀原则** | 给定两个具有相同泛化误差的模型，较简单的模型更可取。 | [[1.机器学习概述（AI增强版）]] |
| **偏差** | 模型预测值与真实值的系统性差距，反映模型本身的拟合能力。 | [[5.集成学习（AI增强版）]] |
| **方差** | 模型对训练数据微小变化的敏感程度，反映模型的稳定性。 | [[5.集成学习（AI增强版）]] |

### 1.2 监督学习算法

#### KNN 算法

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **KNN** | 基于实例的学习算法，通过计算待预测样本与训练样本的距离，找最近的 K 个邻居投票（分类）或取均值（回归）。 | [[2.KNN算法（AI增强版）]] |
| **惰性学习** | 没有显式的训练过程，训练阶段只是存储数据，预测时才计算。 | [[2.KNN算法（AI增强版）]] |
| **欧氏距离** | 最常用直线距离，公式为 √(Σ(xi−yi)²)。 | [[2.KNN算法（AI增强版）]] |
| **曼哈顿距离** | 城市街区距离，公式为 Σ\|xi−yi\|，对异常值更鲁棒。 | [[2.KNN算法（AI增强版）]] |
| **切比雪夫距离** | 各维度最大差值，公式为 max(\|xi−yi\|)。 | [[2.KNN算法（AI增强版）]] |
| **闵可夫斯基距离** | 上述三种距离的通用形式，p=2 为欧氏，p=1 为曼哈顿，p→∞ 为切比雪夫。 | [[2.KNN算法（AI增强版）]] |
| **K 值** | KNN 中近邻的数量，过小易过拟合，过大易欠拟合。 | [[2.KNN算法（AI增强版）]] |
| **归一化** | 将数据映射到 [min, max] 区间（默认 [0,1]），受极值影响大。 | [[2.KNN算法（AI增强版）]] |
| **标准化** | 将数据转换为均值为 0、标准差为 1 的分布，对异常值更鲁棒。 | [[2.KNN算法（AI增强版）]] |

#### 线性回归

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **线性回归** | 有监督回归算法，通过线性方程对一个或多个自变量和因变量的关系建模。 | [[3.线性回归（AI增强版）]] |
| **一元线性回归** | 1 个特征 + 1 个标签，方程为 y = wx + b。 | [[3.线性回归（AI增强版）]] |
| **多元线性回归** | 多个特征 + 1 个标签，方程为 y = W^T X + b。 | [[3.线性回归（AI增强版）]] |
| **权重 (w)** | 特征前的系数，决定该特征对预测结果的影响程度。 | [[3.线性回归（AI增强版）]] |
| **偏置 (b)** | 截距项，独立于特征，用于平移回归线。 | [[3.线性回归（AI增强版）]] |
| **最小二乘法** | 通过最小化误差平方和求解最优参数的解析方法。 | [[3.线性回归（AI增强版）]] |
| **梯度下降** | 沿损失函数负梯度方向迭代更新参数，求解最小值的优化方法。 | [[3.线性回归（AI增强版）]] |
| **学习率** | 控制每次参数更新幅度的超参数，通常取 0.001~0.01。 | [[3.线性回归（AI增强版）]] |
| **全梯度下降 (FGD)** | 使用全部样本计算梯度，稳定但速度慢。 | [[3.线性回归（AI增强版）]] |
| **随机梯度下降 (SGD)** | 随机选 1 个样本计算梯度，速度快但震荡大。 | [[3.线性回归（AI增强版）]] |
| **小批量梯度下降 (mini-batch)** | 随机选小部分样本计算梯度，兼顾速度与稳定性。 | [[3.线性回归（AI增强版）]] |
| **L1 正则化 (Lasso)** | 在损失函数中加 λΣ\|wi\|，可将不重要的特征权重压缩为 0，实现特征选择。 | [[3.线性回归（AI增强版）]] |
| **L2 正则化 (Ridge)** | 在损失函数中加 λΣwi²，将权重压缩到很小，防止过拟合。 | [[3.线性回归（AI增强版）]] |
| **弹性网络 (ElasticNet)** | L1 与 L2 正则化的组合，通过 l1_ratio 控制两者比例。 | [[3.线性回归（AI增强版）]] |

#### 决策树

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **决策树** | 基于 if-else 条件判断的树形监督学习算法，每个内部节点表示一个特征判断。 | [[4.决策树（AI增强版）]] |
| **CART** | 分类与回归树，每次只做二分，分类用基尼指数最小化，回归用平方损失最小化。 | [[4.决策树（AI增强版）]] |
| **信息增益** | 熵减少量，ID3 算法的划分标准，值越大越优先，但偏向特征值多的特征。 | [[4.决策树（AI增强版）]] |
| **信息增益率** | 信息增益除以分裂信息（IV），C4.5 算法使用，克服 ID3 偏向多值特征的问题。 | [[4.决策树（AI增强版）]] |
| **基尼指数** | 衡量数据不纯度，公式为 1−Σpk²，值越小越纯，CART 分类树的默认标准。 | [[4.决策树（AI增强版）]] |
| **预剪枝** | 在树生成过程中提前终止分裂，降低训练时间但可能欠拟合。 | [[4.决策树（AI增强版）]] |
| **后剪枝** | 先生成完整树，再自底向上考察替换为叶节点，泛化性能通常更优。 | [[4.决策树（AI增强版）]] |
| **分类树** | 输出离散类别，叶子节点为出现次数最多的类别。 | [[4.决策树（AI增强版）]] |
| **回归树** | 输出连续值，叶子节点为样本的均值。 | [[4.决策树（AI增强版）]] |
| **特征重要性** | 反映各特征对分裂的贡献度，总和为 1，可用于特征筛选。 | [[4.决策树（AI增强版）]] |

#### 逻辑回归

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **逻辑回归** | 分类算法，将线性回归输出通过 sigmoid 函数映射为 (0,1) 概率值，再设阈值判定类别。 | [[7.逻辑回归（AI增强版）]] |
| **sigmoid 函数** | S 型函数 σ(z)=1/(1+e^(−z))，将实数映射到 (0,1) 区间，用于输出概率。 | [[7.逻辑回归（AI增强版）]] |
| **对数损失 / 交叉熵** | 逻辑回归的损失函数，对错误预测惩罚剧烈，保证凸优化。 | [[7.逻辑回归（AI增强版）]] |
| **决策阈值** | 默认 0.5，概率大于阈值判为正类，降低阈值提高召回率，升高阈值提高精确率。 | [[7.逻辑回归（AI增强版）]] |
| **OvR (One-vs-Rest)** | 多分类策略，每个类对其余所有类训练一个二分类器。 | [[7.逻辑回归（AI增强版）]] |
| **OvO (One-vs-One)** | 多分类策略，每两个类之间训练一个二分类器。 | [[7.逻辑回归（AI增强版）]] |
| **solver** | sklearn 逻辑回归的优化算法选择，如 liblinear、lbfgs、saga 等。 | [[7.逻辑回归（AI增强版）]] |
| **C 参数** | 正则化力度的倒数，C 越大正则化越弱，C 越小正则化越强。 | [[7.逻辑回归（AI增强版）]] |

### 1.3 无监督学习算法

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **K-Means** | 基于质心的划分聚类方法，迭代计算样本到质心的距离并更新质心，直到收敛。 | [[6.聚类学习（AI增强版）]] |
| **质心** | 聚类中心，K-Means 中通过簇内样本取均值更新。 | [[6.聚类学习（AI增强版）]] |
| **K-Means++** | 通过概率化选择初始质心（距离越远概率越大），优化 K-Means 的收敛质量。 | [[6.聚类学习（AI增强版）]] |
| **DBSCAN** | 基于密度的聚类算法，通过 eps 邻域和最小样本数发现任意形状的簇。 | [[6.聚类学习（AI增强版）]] |
| **核心点** | DBSCAN 中 eps 邻域内样本数不少于 min_samples 的点。 | [[6.聚类学习（AI增强版）]] |
| **边界点** | DBSCAN 中落在核心点邻域内但自身邻域样本不足的点。 | [[6.聚类学习（AI增强版）]] |
| **噪声点** | DBSCAN 中不属于任何核心点邻域的点，即离群点。 | [[6.聚类学习（AI增强版）]] |
| **层次聚类** | 对数据逐层划分（凝聚或分裂），直到达到指定类别数。 | [[6.聚类学习（AI增强版）]] |
| **硬聚类** | 每个样本只属于一个簇，如 K-Means。 | [[6.聚类学习（AI增强版）]] |
| **软聚类** | 每个样本以一定概率属于多个簇，如高斯混合模型。 | [[6.聚类学习（AI增强版）]] |
| **肘部法则** | 通过绘制不同 K 值下的 SSE 曲线，找到下降速度突然变缓的拐点确定最优 K。 | [[6.聚类学习（AI增强版）]] |
| **轮廓系数** | 衡量簇内紧密度 vs 簇间分离度，范围 [−1,1]，越接近 1 聚类效果越好。 | [[6.聚类学习（AI增强版）]] |
| **CH 指数** | 簇间方差与簇内方差的比值，越大越好，计算效率高适合大数据集。 | [[6.聚类学习（AI增强版）]] |
| **SSE** | 簇内样本到质心的误差平方和，越小表示簇越紧凑。 | [[6.聚类学习（AI增强版）]] |

### 1.4 集成学习

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **集成学习** | 通过组合多个弱学习器形成强学习器，核心思想"三个臭皮匠，赛过诸葛亮"。 | [[5.集成学习（AI增强版）]] |
| **弱学习器 / 基学习器** | 参与组合的单个模型，如单棵决策树。 | [[5.集成学习（AI增强版）]] |
| **强学习器** | 组合后的集成模型，精度高于单个弱学习器。 | [[5.集成学习（AI增强版）]] |
| **Bagging** | 并行训练多模型，有放回抽样产生不同训练集，平权投票降低方差。 | [[5.集成学习（AI增强版）]] |
| **Boosting** | 串行训练，后一个模型关注前一个模型的错误，加权结合降低偏差。 | [[5.集成学习（AI增强版）]] |
| **Bootstrap** | 有放回抽样，每棵树约使用 63.2% 的原始样本，生成多样性训练子集。 | [[5.集成学习（AI增强版）]] |
| **随机森林** | 基于 Bagging+决策树的集成算法，引入数据随机和特征随机双重随机性。 | [[5.集成学习（AI增强版）]] |
| **OOB (袋外数据)** | Bootstrap 未抽到的约 36.8% 样本，可用于无偏估计模型性能。 | [[5.集成学习（AI增强版）]] |
| **AdaBoost** | 自适应提升，通过提高前一轮错分样本的权重迭代训练强分类器。 | [[5.集成学习（AI增强版）]] |
| **GBDT** | 梯度提升决策树，用损失函数的负梯度拟合残差，串行构建多棵树。 | [[5.集成学习（AI增强版）]] |
| **残差** | 真实值与预测值的差，GBDT 每一轮拟合上一轮残差。 | [[5.集成学习（AI增强版）]] |
| **shrinkage (学习率)** | GBDT 中控制每棵树贡献的衰减系数，防止过拟合。 | [[5.集成学习（AI增强版）]] |
| **XGBoost** | 极致梯度提升，在 GBDT 基础上加入二阶泰勒展开、正则化和并行优化。 | [[5.集成学习（AI增强版）]] |
| **Stacking** | 用另一个元模型组合多个基学习器的预测结果，形成更高层集成。 | [[5.集成学习（AI增强版）]] |
| **偏差-方差分解** | 将模型误差分解为偏差（欠拟合）和方差（过拟合），指导算法选择。 | [[5.集成学习（AI增强版）]] |

### 1.5 评估与优化

| 概念 | 一句话定义 | 来源 |
|------|-----------|------|
| **交叉验证** | 将训练集划分为 n 份，轮流用其中 1 份做验证集，训练 n 次取平均得分。 | [[2.KNN算法（AI增强版）]] |
| **网格搜索** | 自动遍历超参数的所有组合，每组都用交叉验证评估，选出最优参数组合。 | [[2.KNN算法（AI增强版）]] |
| **随机搜索** | 在超参数空间中随机采样组合，适合参数维度高时替代网格搜索。 | [[2.KNN算法（AI增强版）]] |
| **超参数** | 模型训练前需手动设定的参数（如 K 值、学习率），与训练自动学习的参数不同。 | [[2.KNN算法（AI增强版）]] |
| **混淆矩阵** | TP/FP/TN/FN 四格表，全面评估分类结果的分布。 | [[2.KNN算法（AI增强版）]] |
| **TP (真正例)** | 真实为正例且预测为正例的样本数。 | [[2.KNN算法（AI增强版）]] |
| **FP (假正例)** | 真实为反例但预测为正例的样本数（误报）。 | [[2.KNN算法（AI增强版）]] |
| **TN (真反例)** | 真实为反例且预测为反例的样本数。 | [[2.KNN算法（AI增强版）]] |
| **FN (假反例)** | 真实为正例但预测为反例的样本数（漏报）。 | [[2.KNN算法（AI增强版）]] |
| **准确率** | (TP+TN)/(TP+FP+TN+FN)，预测正确的样本占总样本的比例。 | [[2.KNN算法（AI增强版）]] |
| **精确率** | TP/(TP+FP)，预测为正例中真正例的比例（查准率）。 | [[2.KNN算法（AI增强版）]] |
| **召回率** | TP/(TP+FN)，真实正例中被预测出的比例（查全率）。 | [[2.KNN算法（AI增强版）]] |
| **F1-Score** | 精确率与召回率的调和平均，兼顾两者，类别不均衡时优先参考。 | [[2.KNN算法（AI增强版）]] |
| **ROC 曲线** | 以 TPR（召回率）为纵轴、FPR 为横轴，绘制不同阈值下的性能曲线。 | [[2.KNN算法（AI增强版）]] |
| **AUC** | ROC 曲线下面积，衡量分类器综合区分能力，越接近 1 越好。 | [[2.KNN算法（AI增强版）]] |
| **MSE** | 均方误差，预测值与真实值差值的平方平均，对异常值敏感。 | [[3.线性回归（AI增强版）]] |
| **RMSE** | 均方根误差，√MSE，与原始数据同量纲更直观。 | [[3.线性回归（AI增强版）]] |
| **MAE** | 平均绝对误差，预测值与真实值差值的绝对平均，对异常值鲁棒。 | [[3.线性回归（AI增强版）]] |
| **R² 决定系数** | 衡量回归模型拟合优度，1−MSE/Var(y)，越接近 1 越好，可为负。 | [[3.线性回归（AI增强版）]] |
| **正则化** | 在损失函数中添加惩罚项，限制模型复杂度，防止过拟合。 | [[3.线性回归（AI增强版）]] |
| **数据泄露** | 测试集信息泄露到训练过程，如用全部数据标准化后再划分训练/测试。 | [[2.KNN算法（AI增强版）]] |

## 二、学习路径

> 建议按以下三阶段顺序学习，后一阶段建立在前一阶段的基础之上。

### 阶段 1：基础认知（建立框架）

**目标**：建立机器学习的整体认知框架，掌握评估模型的通用语言。

**必读**：[[1.机器学习概述（AI增强版）]]
- 必须掌握：核心术语（样本/特征/标签/训练集/测试集）
- 必须掌握：四大学习范式（监督/无监督/半监督/强化）的定义与区别
- 必须掌握：建模六步法与特征工程五大手段
- 必须掌握：欠拟合 vs 过拟合的判断与应对思路
- **预计时间**：2~3 小时

### 阶段 2：核心算法（逐个攻克）

**目标**：掌握最常用的监督与无监督算法，理解核心思想和适用边界。

**2.1 相似性算法入门** → [[2.KNN算法（AI增强版）]]
- 前置知识：距离的概念
- 核心概念：K 值选择、距离度量、特征归一化/标准化
- 评估指标：混淆矩阵、精确率、召回率、F1-Score
- 调优方法：交叉验证、网格搜索
- **预计时间**：2~3 小时

**2.2 参数化模型基础** → [[3.线性回归（AI增强版）]]
- 前置知识：KNN、导数基础
- 核心概念：损失函数（MSE/MAE）、梯度下降、学习率、正则化（L1/L2）
- 评估指标：R²、MSE、RMSE、MAE
- **预计时间**：3~4 小时

**2.3 分类入门** → [[7.逻辑回归（AI增强版）]]
- 前置知识：线性回归（它是线性回归+sigmoid 的扩展）
- 核心概念：sigmoid 函数、对数损失、概率输出、决策阈值
- **预计时间**：2~3 小时

**2.4 非线性模型入门** → [[4.决策树（AI增强版）]]
- 前置知识：信息论基础（熵的概念）
- 核心概念：信息增益/增益率/基尼指数、CART、预剪枝/后剪枝
- **预计时间**：3~4 小时

**2.5 无监督学习** → [[6.聚类学习（AI增强版）]]
- 前置知识：KNN 中的距离度量、线性回归中的均值计算
- 核心概念：K-Means 迭代流程、K 值确定（肘部法则/轮廓系数）、DBSCAN 密度思想
- **预计时间**：2~3 小时

### 阶段 3：进阶提升（竞赛实战）

**目标**：理解"组合多个弱模型变强"的思想，掌握竞赛级算法。

**必读**：[[5.集成学习（AI增强版）]]
- 前置知识：**决策树**（必须！所有集成学习几乎都以决策树为基学习器）
- 建议顺序：Bagging（随机森林）→ Boosting（AdaBoost→GBDT→XGBoost）
- 核心概念：Bootstrap、Bagging vs Boosting、残差、偏差-方差分解、OOB
- **预计时间**：4~5 小时

**概念依赖关系图**：

```
机器学习概述（评估指标、过拟合概念）
    │
    ├──→ KNN（相似性、距离、归一化）
    │       └──→ 聚类学习（K-Means 也用距离和均值）
    │
    ├──→ 线性回归（损失函数、梯度下降、正则化）
    │       └──→ 逻辑回归（+ sigmoid = 分类）
    │
    └──→ 决策树（信息论、递归划分、剪枝）
            └──→ 集成学习（Bagging/Boosting 以决策树为基）
                    ├──→ 随机森林（Bagging + 决策树）
                    ├──→ AdaBoost（Boosting + 决策树）
                    ├──→ GBDT（梯度 + Boosting + 决策树）
                    └──→ XGBoost（优化版 GBDT）
```

## 三、sklearn API 速查

### 通用模式

所有 sklearn 估计器遵循统一接口：

```python
# 1. 导入模型
from sklearn.xxx import ModelName

# 2. 实例化（设置超参数）
model = ModelName(param1=value1, param2=value2)

# 3. 训练
model.fit(X_train, y_train)

# 4. 预测
y_pred = model.predict(X_test)        # 预测标签
y_prob = model.predict_proba(X_test)  # 预测概率（分类器）

# 5. 评估
score = model.score(X_test, y_test)   # 默认：分类=准确率，回归=R²
```

### 各算法核心类与参数

| 算法 | 核心类 | 关键参数 | 适用场景 |
|------|--------|----------|----------|
| **KNN 分类** | `sklearn.neighbors.KNeighborsClassifier` | `n_neighbors`（K 值）、`weights`（uniform/distance）、`metric`（距离类型） | 小样本、低维度、可解释性要求高 |
| **KNN 回归** | `sklearn.neighbors.KNeighborsRegressor` | `n_neighbors`、`weights`、`metric` | 连续值预测、局部平滑关系 |
| **线性回归** | `sklearn.linear_model.LinearRegression` | `fit_intercept`（是否计算截距） | 小到中等规模数据、特征维度不过高 |
| **SGD 回归** | `sklearn.linear_model.SGDRegressor` | `learning_rate`、`eta0`、`penalty`、`alpha` | 大规模数据、在线学习 |
| **Lasso (L1)** | `sklearn.linear_model.Lasso` | `alpha`（正则化强度） | 需要特征选择的回归任务 |
| **Ridge (L2)** | `sklearn.linear_model.Ridge` | `alpha`（正则化强度） | 多重共线性、防止过拟合 |
| **逻辑回归** | `sklearn.linear_model.LogisticRegression` | `C`（正则化倒数）、`penalty`（l1/l2/elasticnet）、`solver`（优化器） | 二分类、概率输出需求 |
| **决策树分类** | `sklearn.tree.DecisionTreeClassifier` | `criterion`（gini/entropy）、`max_depth`、`min_samples_split` | 特征含义明确、需要强可解释性 |
| **决策树回归** | `sklearn.tree.DecisionTreeRegressor` | `criterion`（squared_error）、`max_depth`、`min_samples_split` | 非线性回归、需要可解释性 |
| **随机森林分类** | `sklearn.ensemble.RandomForestClassifier` | `n_estimators`（树的数量）、`max_depth`、`max_features`、`bootstrap` | 通用分类、特征重要性分析 |
| **随机森林回归** | `sklearn.ensemble.RandomForestRegressor` | `n_estimators`、`max_depth`、`max_features` | 通用回归、抗过拟合 |
| **AdaBoost 分类** | `sklearn.ensemble.AdaBoostClassifier` | `n_estimators`、`learning_rate` | 提升简单模型性能 |
| **GBDT 分类** | `sklearn.ensemble.GradientBoostingClassifier` | `n_estimators`、`learning_rate`、`max_depth` | 高精度分类需求 |
| **GBDT 回归** | `sklearn.ensemble.GradientBoostingRegressor` | `n_estimators`、`learning_rate`、`max_depth` | 高精度回归需求 |
| **XGBoost 分类** | `xgboost.XGBClassifier` | `n_estimators`、`max_depth`、`learning_rate`、`reg_alpha`、`reg_lambda` | 表格数据竞赛首选 |
| **XGBoost 回归** | `xgboost.XGBRegressor` | `n_estimators`、`max_depth`、`learning_rate` | 表格数据竞赛首选 |
| **K-Means** | `sklearn.cluster.KMeans` | `n_clusters`（K 值）、`init`（k-means++/random）、`random_state` | 球形簇、已知大致类别数 |
| **DBSCAN** | `sklearn.cluster.DBSCAN` | `eps`（邻域半径）、`min_samples` | 非球形簇、噪声较多 |
| **Mini-Batch K-Means** | `sklearn.cluster.MiniBatchKMeans` | `n_clusters`、`batch_size` | 大规模数据聚类（>10万样本） |

## 四、FAQ（高频避坑指南）

| 问题 | 答案 | 详细位置 |
|------|------|----------|
| **KNN 的 K 值怎么选？** | 交叉验证/网格搜索，K 过小过拟合、过大欠拟合，常用 3~10。 | [[2.KNN算法（AI增强版）]] |
| **逻辑回归是回归还是分类？** | **分类算法**，只是名字带"回归"（因为用了线性回归的输出结构）。 | [[7.逻辑回归（AI增强版）]] |
| **决策树三种算法怎么选？** | ID3（信息增益，已过时）→ C4.5（信息增益率）→ **CART**（基尼指数，sklearn 默认最常用）。 | [[4.决策树（AI增强版）]] |
| **随机森林为什么不容易过拟合？** | Bagging+随机特征引入多样性，投票平均降低方差，即使单棵树过拟合也会被稀释。 | [[5.集成学习（AI增强版）]] |
| **XGBoost 和 GBDT 的区别？** | XGBoost = GBDT + 二阶泰勒展开 + 正则化 + 并行优化 + 缺失值处理 + 特征子采样。 | [[5.集成学习（AI增强版）]] |
| **K-Means 的 K 不知道怎么办？** | **肘部法则**看 SSE 拐点，或**轮廓系数**自动评估，也可尝试 DBSCAN 无需预设 K。 | [[6.聚类学习（AI增强版）]] |
| **归一化是必须的吗？** | KNN、K-Means、神经网络、SGD 等距离/梯度类算法**必须做**；树模型不需要。 | [[2.KNN算法（AI增强版）]] |
| **正则化 L1 和 L2 怎么选？** | L1 产生稀疏解（特征选择），L2 限制权重大小（防止过拟合），不确定时用 ElasticNet。 | [[3.线性回归（AI增强版）]] |
| **为什么测试集只能用 transform？** | 测试集如果用 fit_transform 会重新计算均值/标准差，导致**数据泄露**，必须用训练集参数转换。 | [[2.KNN算法（AI增强版）]] |
| **数据类别不均衡怎么办？** | 优先看 F1-Score 而非准确率；可设置 `class_weight='balanced'`；或用采样方法。 | [[2.KNN算法（AI增强版）]] |
| **梯度下降学习率怎么设？** | 通常从 0.001~0.01 开始尝试，太小收敛慢，太大震荡或发散。 | [[3.线性回归（AI增强版）]] |
| **决策树深度不设限制会怎样？** | 容易严重过拟合，训练集准确率 100% 但测试集很差，必须设 `max_depth` 或做剪枝。 | [[4.决策树（AI增强版）]] |
| **Boosting 为什么对噪声敏感？** | 噪声样本会被反复错分，导致权重不断放大，最终过拟合。 | [[5.集成学习（AI增强版）]] |
| **线性回归假设不成立怎么办？** | 若特征与目标非线性关系，可添加多项式特征，或换用决策树/神经网络。 | [[3.线性回归（AI增强版）]] |
| **逻辑回归的 solver 怎么选？** | 小数据用 liblinear，大数据用 saga；saga 是唯一同时支持 L1/L2/ElasticNet 的求解器。 | [[7.逻辑回归（AI增强版）]] |
| **DBSCAN 的 eps 怎么选？** | 可通过 k-距离图（k= min_samples−1）找拐点，或使用网格搜索结合轮廓系数评估。 | [[6.聚类学习（AI增强版）]] |
| **特征工程中，提取和选择的区别？** | 提取是从原始数据构造新特征（如 TF-IDF），选择是从已有特征中挑选子集。 | [[1.机器学习概述（AI增强版）]] |
| **随机森林的 n_estimators 设多少？** | 常用 100~500，越多越稳定但训练越慢，边际收益递减。 | [[5.集成学习（AI增强版）]] |
| **GBDT 为什么训练慢？** | 串行训练，每棵树依赖前一棵的残差，无法并行；可用 XGBoost/LightGBM 加速。 | [[5.集成学习（AI增强版）]] |
| **半监督学习和监督学习的本质区别？** | 半监督学习用少量标签+大量无标签数据训练，大幅降低标注成本。 | [[1.机器学习概述（AI增强版）]] |
