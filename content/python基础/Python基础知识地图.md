## 四、FAQ（高频避坑问答）

> 以下问题从 15 篇笔记的「避坑指南 & 易错对比」中提取，覆盖新手最常见错误。

### Q1：`input()` 获取的数字能直接做数学运算吗？
**不能。** `input()` 返回的永远是字符串，必须先用 `int()` 或 `float()` 转换后再进行数学运算。详见 [[day01.Python基础入门（AI增强版）]] 和 [[day02.类型转换与控制流（AI增强版）]]。

### Q2：`=` 和 `==` 有什么区别？
- `=` 是**赋值运算符**，将右边的值赋给左边的变量。
- `==` 是**比较运算符**，判断两边的值是否相等。
- `if a = 5:` 会报 `SyntaxError`！详见 [[day02.类型转换与控制流（AI增强版）]]。

### Q3：`/` 和 `//` 有什么区别？
- `/` 是**浮点除法**，永远返回浮点数（如 `5 / 2` → `2.5`）。
- `//` 是**整除（地板除）**，返回整数部分（如 `5 // 2` → `2`）。详见 [[day02.类型转换与控制流（AI增强版）]]。

### Q4：为什么遍历列表时删除元素会出错？
因为在遍历过程中修改列表会导致索引错乱，部分元素被跳过。正确做法：遍历副本 `for n in nums[:]:`，或用列表推导式重建列表。详见 [[day02.类型转换与控制流（AI增强版）]] 和 [[day03.字符串与列表操作（AI增强版）]]。

### Q5：`sort()` 和 `sorted()` 有什么区别？
- `sort()` **原地排序**，修改原列表，返回 `None`。
- `sorted()` **返回新列表**，不修改原列表。注意不要写 `nums = nums.sort()`（会变成 `None`）。详见 [[day03.字符串与列表操作（AI增强版）]]。

### Q6：浅拷贝和深拷贝有什么区别？
- **浅拷贝**（`copy()`、`[:]`）：只复制最外层容器，内部对象共享引用。
- **深拷贝**（`copy.deepcopy()`）：递归复制所有层级，生成完全独立的对象。嵌套列表必须使用深拷贝。详见 [[day03.字符串与列表操作（AI增强版）]] 和 [[day08.闭包与装饰器（AI增强版）]]。

### Q7：为什么不能用可变对象作为默认参数？
因为默认参数在函数定义时只创建一次，后续调用共享同一个对象。如 `def f(lst=[]):` 会导致列表累积。正确做法：`def f(lst=None):`，函数内部判断 `if lst is None: lst = []`。详见 [[day03.字符串与列表操作（AI增强版）]] 和 [[day05.函数与文件异常处理（AI增强版）]]。

### Q8：单元素元组怎么写？
必须加逗号：`t = (1,)`。`(1)` 会被解析为整数 `1` 而非元组。详见 [[day04.元组集合字典与推导式（AI增强版）]]。

### Q9：`{}` 是空集合还是空字典？
`{}` 是**空字典**；空集合必须写 `set()`。详见 [[day04.元组集合字典与推导式（AI增强版）]]。

### Q10：`dict[key]` 和 `dict.get(key)` 有什么区别？
- `dict[key]`：键不存在时抛出 `KeyError`。
- `dict.get(key)`：键不存在时返回 `None`（或指定的默认值），更安全。详见 [[day04.元组集合字典与推导式（AI增强版）]]。

### Q11：`global` 和 `nonlocal` 有什么区别？
- `global`：修改**模块级全局变量**。
- `nonlocal`：修改**嵌套函数中最近一层外层**的非全局变量。详见 [[day05.函数与文件异常处理（AI增强版）]] 和 [[day08.闭包与装饰器（AI增强版）]]。

### Q12：为什么文件操作推荐用 `with open(...)`？
`with` 语句会自动调用文件的 `__enter__` 和 `__exit__` 方法，即使发生异常也能确保文件被正确关闭，避免资源泄漏。详见 [[day05.函数与文件异常处理（AI增强版）]] 和 [[day06.环境管理与HTTP基础（AI增强版）]]。

### Q13：`self` 和 `cls` 有什么区别？
- `self`：实例方法的第一个参数，代表**实例本身**。
- `cls`：类方法的第一个参数，代表**类本身**（由 `@classmethod` 装饰）。详见 [[day07.面向对象编程（AI增强版）]]。

### Q14：实例属性和类属性有什么区别？
- **实例属性**：用 `self.xxx` 定义，每个实例独立拥有。
- **类属性**：在类体中直接定义，所有实例共享。修改类属性会影响所有实例。详见 [[day07.面向对象编程（AI增强版）]]。

### Q15：`__str__` 和 `__repr__` 有什么区别？
- `__str__`：面向**用户**的友好字符串表示（`print()` 调用）。
- `__repr__`：面向**开发者**的精确字符串表示（交互式输出调用），通常应能 `eval()` 还原。详见 [[day07.面向对象编程（AI增强版）]]。

### Q16：装饰器为什么要加 `@wraps`？
不加 `@wraps` 时，被装饰函数的 `__name__` 和 `__doc__` 会被 `wrapper` 函数覆盖。`@wraps(func)` 保留原函数的元数据，是最佳实践。详见 [[day08.闭包与装饰器（AI增强版）]]。

### Q17：TCP 和 UDP 有什么区别？
- **TCP**：面向连接、可靠传输、有序到达，适合文件传输、网页浏览。
- **UDP**：无连接、高效、不保证送达，适合视频直播、DNS 查询。详见 [[day09.网络编程与JSON（AI增强版）]]。

### Q18：`json.dumps` 和 `json.dump` 有什么区别？
- `json.dumps()`：将 Python 对象序列化为 **JSON 字符串** 并返回。
- `json.dump()`：将 Python 对象直接**写入文件对象**。同理 `loads` vs `load`。详见 [[day09.网络编程与JSON（AI增强版）]]。

### Q19：迭代器和生成器有什么区别？
- **迭代器**：实现 `__iter__()` 和 `__next__()` 的类，代码量较多。
- **生成器**：使用 `yield` 的函数，更简洁，自动保存状态。两者都支持惰性求值。详见 [[day10.高阶特性与并发编程（AI增强版）]]。

### Q20：进程、线程、协程有什么区别？
- **进程**：操作系统资源分配的最小单位，拥有独立内存空间，适合 CPU 密集型任务。
- **线程**：CPU 调度的最小单位，同一进程内共享内存，受 GIL 限制。
- **协程**：单线程内的用户态并发，适合 I/O 密集型高并发场景。详见 [[day10.高阶特性与并发编程（AI增强版）]]。

### Q21：NumPy 中 `*` 和 `@` 有什么区别？
- `*` 是**逐元素相乘**（Hadamard 积）。
- `@` 或 `np.dot` 是**矩阵乘法**（行列点积）。两者完全不同，维度不匹配时会报错。详见 [[day13.NumPy与Pandas数据分析（AI增强版）]]。

### Q22：Pandas 中 `loc` 和 `iloc` 有什么区别？
- `loc`：按**标签索引**，切片包含终点（如 `df.loc[0:2]` 包含第 2 行）。
- `iloc`：按**位置索引**，切片不含终点（如 `df.iloc[0:2]` 只包含第 0、1 行）。详见 [[day13.NumPy与Pandas数据分析（AI增强版）]]。

### Q23：AI、ML、DL 是什么关系？
- **AI（人工智能）** 范围最广泛。
- **ML（机器学习）** 是 AI 的子集，专注数据驱动学习。
- **DL（深度学习）** 是 ML 的子集，专注神经网络。
关系：AI ⊃ ML ⊃ DL。详见 [[day15.AI概述与前沿展望（AI增强版）]]。

### Q24：Transformer 为什么重要？
Transformer 用自注意力机制取代 RNN/CNN，实现了完全并行、长距离依赖和强可扩展性，已成为 NLP、视觉和多模态的通用架构，是大语言模型（LLM）的基础。详见 [[day15.AI概述与前沿展望（AI增强版）]]。
